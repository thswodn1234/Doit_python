### for문을 두번 사용해 19단 곱셈표 출력하기

### 문자열 포매팅으로 print() 함수 더 편하게 사용하기

### lambda로 간단하게 함수 만들기
함수를 만드는 기본 명령어는 def문 

lambda는 단 한 줄로 함수를 만들어내는 강력함을 가지고 있음

ex) y = 3x 함수 만들기
```python
>>> y = lambda x : 3 * x
>>> y(12)
36
```

a, b라는 두 개의 수를 입력받아 더하는 함수
```python
>>> add = lambda a, b : a + b
>>> add(2,3)
5
```

긴 글(문자열)은 작은따옴표(또는 큰따옴표) 세 개(```)를 사용해 변수에 저장 가능

문자열 슬라이싱이란 문자열에서 범위를 지정해 선택한 부분만 가져오는 것

컴퓨터는 공백도 글자로 인식

ex) 어떤 문자열이든 열 글자 까지 출력하는 함수
```python
>>> littleprince = '''여섯 살 적에 나는 '체험한 ...'''
>>> short = lambda x : x[:10]
>>> short(littleprince)
'여섯 살 적에 나는'
```

### lambda 주의점
lambda를 남발하면 가독성 떨어짐 
다른 개발자가 읽기 힘듬

### def 문으로 함수 만들기

### input() 함수로 사용자에게 입력값 받기
input()함수는 사용자에게 필요한 값을 직접 전달받아서 객체에 입력하는 역할을 함

input()함수 사용법
```python
input('사용자에게 입력받을 값을 설명하는 내용')
```

```python
>>> a = input('숫자를 하나 입력해 보세요: ')
숫자를 하나 입력해 보세요: 23
>>> a
'23'
>>> type(a)
<class 'str'>
>>> a = int(a)
>>> type(a)
<class 'int'>
>>> a = float(a)
>>> type(a)
<class 'float'>
```

input()함수는 사용자가 입력한 값을 문자(str)로 저장

부동소수점(floating point)이란 정확한 소수점 자리를 지정하지 않고,

어떤 객체의 유형을 정수로 하려면 int() 함수를
실수로 바꾸고 싶으면 float() 함수를 사용

```python
>>> c = 17 / 7
>>> c
2.4285714285714284
>>> type(c)

>>> c
```
